Installation
==============

Plates is available on [Packagist](https://packagist.org/packages/league/plates) and can be installed using [Composer](https://getcomposer.org/):

~~~.language-javascript
{
    "require": {
        "league/plates": "1.1.*"
    }
}
~~~

~~~.language-php
<?php

// Include the Composer autoloader
require 'vendor/autoload.php';
~~~
